== Installation ==

1. Upload 'JDs_portfolio' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "JDs Portfolio" and create your first Portfolio to show your Work!
